# SVBurger Automation Framework

Automation testing framework for SVBurger web application using Playwright and Python.

## Technologies used

- Python
- Playwright
- Page Object Model (POM)
- PyCharm
- Git & GitHub

## Project structure

pages/ → Page Object classes  
tests/ → Test cases  
utils/ → BrowserFactory and helper classes  
requirements/ → dependencies  
SVBurger.py → main framework entry point  

## How to run tests

Install dependencies:
pip install -r requirements/requirements.txt

Run tests:
pytest tests/

## Framework features

- Page Object Model (POM)
- Playwright browser automation
- Reusable BrowserFactory
- Scalable test framework structure
